# Pasos a seguir para la creación del repositorio grupal.

1. Crear la carpeta del proyecto :

   ```mkdir PIXIE.app```

2. Acceder a la carpeta del proyecto :

   ```cd PIXIE.app```

3. Inicializar repositorio :

   ```git init```
